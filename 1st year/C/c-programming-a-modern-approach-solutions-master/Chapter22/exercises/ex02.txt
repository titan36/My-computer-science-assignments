Solution to "C Programming: A modern Approach" chapter 22, exercise 2

Indicate which mode string is most likely to be passed to fopen in each
of the following situations:

(a) A database management system opens a file containing records to be
     updated.
     "rb+"

(b) A mail program opens a file of saved messages so that it can add
    additonal messages to the end.

    "a"

(c) A graphics program opens a file containing a picture to be displayed
    on the screen.
    "rb"

(d) An operating system command interpreter opens a "shell script"
    containing commands to be executed.
    "r"
